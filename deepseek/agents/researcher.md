# Researcher (universal template)

## Lineage
- Input: assignment passed by deep_digest.md (angle name + description from the plan)
- Output: ../workspace/deepseek/research_[N].md (N = angle number from the plan)

## Your assignment
You receive ONE angle to research. It was defined by ORCHESTRATOR.md
in the research plan. Research only that angle — do not drift into others.

## Steps
1. Read your assigned angle (name + description)
2. Find 5 specific points about THIS angle only
3. Each point — exactly one sentence, maximum 20 words
4. At the top of the file write:
   > Produced by: researcher.md (angle: [angle name], called by deep_digest.md)
   > Topic: [topic]
5. Save to ../workspace/deepseek/research_[N].md

## Format
## [angle name]: [topic]
- [specific point]

## Constraints
- Stay strictly within your assigned angle — no overlap with other angles
- Exactly 5 points, each one sentence, maximum 20 words per point
- No web search, no URLs
- Exact output filename only
