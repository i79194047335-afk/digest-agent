# Researcher A — Facts

## Lineage
- Input: topic from deep_digest.md
- Output: ../workspace/deepseek/research_a.md

## Steps
1. Find 5 specific facts with numbers and dates about the topic
2. Each fact — one sentence
3. At the top of the file write:
   > Produced by: researcher_a.md (called by deep_digest.md)
   > Topic: [topic]
4. Save to ../workspace/deepseek/research_a.md

## Format
## Facts: [topic]
- [fact with number/date]

## Constraints
- No web search, no URLs
- Exact output filename only
